## 标准库函数



### collections.namedtuple

这是标准库 `collections` 的一个函数 `namedtupe`, 可以用来容易地创建一个简单的类, 示例如下

```python
from collections import namedtuple

# 创建一个Dog类
Dog = namedtuple('Dog', ['name', 'weight', 'owner'])
rex = Dog('rex', 30, 'Bob')
print(rex)
```
